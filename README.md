# BatBot - Telegram бот, имитирующий Бэтмена

## О проекте

BatBot - это простой Telegram бот, который имитирует общение с Бэтменом. Бот отвечает на приветствия и делится цитатами, характерными для этого персонажа.

## Технологии

- **Язык программирования**: JavaScript (Node.js)
- **Библиотека**: node-telegram-bot-api
- **Платформа**: Telegram Bot API

## Требования для запуска

- Установленный Node.js
- Токен Telegram бота, полученный от @BotFather

## Установка и запуск

1. Склонируйте репозиторий или создайте файл с кодом бота

2. Установите необходимые зависимости:
```bash
npm install node-telegram-bot-api
```

3. Получите токен бота у @BotFather в Telegram

4. Вставьте полученный токен в код:
```javascript
const token = 'ВАШ_ТОКЕН_ЗДЕСЬ';
```

5. Запустите бота:
```bash
node bot.js
```

## Функциональность

Бот реагирует на следующие сообщения пользователя:

| Команда или фраза | Ответ бота |
|-------------------|------------|
| Привет (в начале сообщения) | Привет, гражданский |
| пока (в любом месте сообщения) | прощай, а я пойду на поиски джокера |
| скажи цитату | жизнь ухудшается перед тем, как наладиться. |
| цитата о правде | порой правда недостаточно хороша, порой люди заслуживают большего, порой люди заслуживают, чтобы их вера вознаграждалась. |
| у тебя есть друзья? | друзья для меня — роскошь. |
| ты любил кого-то? | Однажды я полюбил женщину-кошку, однако она не ответила мне взаимностью. |
| дашь свою маску? | эта маска — не для тебя, а для того, чтобы защитить твоих близких. |

## Дополнительные возможности

### Команда /echo

Бот поддерживает команду `/echo`, которая повторяет текст, введенный после команды.

**Пример:**
```
/echo Привет, мир!
```
Ответ бота: `Привет, мир!`

## Структура кода

```javascript
const TelegramBot = require('node-telegram-bot-api');

// Инициализация бота с токеном и режимом polling
const bot = new TelegramBot(token, {polling: true});

// Обработка команды /echo
bot.onText(/\/echo (.+)/, (msg, match) => {
  const chatId = msg.chat.id;
  const resp = match[1];
  bot.sendMessage(chatId, resp);
});

// Обработка текстовых сообщений
bot.on('message', (msg) => {
  // Проверка сообщений и отправка соответствующих ответов
});
```

## Примечания

- Бот работает в режиме polling, что означает постоянный опрос сервера Telegram на наличие новых сообщений
- Все сравнения сообщений регистронезависимы (преобразуются в нижний регистр)
- Бот не сохраняет историю сообщений и не использует базу данных
